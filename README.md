# 🏨 Hotel Management System

A simple console-based Hotel Management System written in C++ for managing room bookings, check-ins, check-outs, and data storage.

## 💻 Features

- View room status
- Book a room
- Check out a room
- Search room booking info
- Save booking & checkout info to text files

## 🧠 Technologies Used

- C++ (Basic I/O, functions, file handling)

## 🚀 How to Run

1. Open this project in any C++ IDE (e.g., CodeBlocks, Dev C++, or Visual Studio)
2. Compile and run `hotel_management.cpp`
3. Follow the on-screen menu
